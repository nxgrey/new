## ⚗️ Installation

Clone the project

```bash
  git clone https://github.com/rupam-seal/CardPrinting.git
```

<br>

Go to the project directory

```bash
  cd CardPrinting
```

<br>

Install Dependencies

```bash
  pip install -r requirements.txt
```

<br>

Start the server

```bash
  run CardPrinting.py
```
